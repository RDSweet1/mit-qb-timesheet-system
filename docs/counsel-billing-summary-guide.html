Counsel Billing Summary

User Guide — MIT Consulting Timesheet System

Overview

The Counsel Billing Summary generates a comprehensive billing report for opposing counsel in litigation matters. It pulls real invoice data and time records from QuickBooks Online, strips all notes and descriptions (work product protection), and presents a clean summary of professional services rendered.

Key Feature: Notes and descriptions are automatically removed from the report. Only dates, professionals, service categories, clock times, rates, and amounts are shown — protecting attorney work product while providing full billing transparency.

Accessing the Report

  1. Navigate to the Reports tab in the main navigation bar
  2. Click the Counsel Billing Summary tab at the top of the page
  3. You'll see the configuration panel with customer selection and date range options

Generating a Report

Step 1: Select the Customer

Use the Customer dropdown to select the client/matter you want to generate the report for. All active customers from QuickBooks are listed alphabetically.

Step 2: Set the Date Range

Choose a preset or set custom dates:

PresetDescriptionBest For
Show All DataFrom 2020 to todayFull engagement history (default)
This QuarterCurrent calendar quarterQuarterly billing reviews
Year to DateJan 1 to todayAnnual summaries
Last YearFull previous calendar yearYear-end reporting
Custom (From/To)Any date rangeSpecific litigation periods

Step 3: Generate

Click Generate Report. The system will:

  1. Query QuickBooks Online for all invoices in the date range
  2. Pull time entries from the Supabase database (includes Workforce clock times)
  3. Fill gaps with QB Online TimeActivity records for older periods
  4. Merge data, deduplicate, and run duration sanity checks
  5. Render the report inline (typically 15-25 seconds for full engagement)
Note: For large date ranges (2+ years), the report may take up to 30 seconds to generate. This is normal — it's querying live QB Online data.

Reading the Report

Summary Cards

Four key metrics appear at the top:

MetricDescription
Total InvoicedSum of all QB invoices in the date range
OutstandingUnpaid invoice balance
Total HoursSum of all time entries
InvoicesNumber of invoices in the period

Invoice Summary Table

Lists every invoice with its number, date, paid/open status, and amount. The total row shows the aggregate.

Invoice Detail Blocks

Each invoice expands to show:

  • QB Line Items — The actual billing lines from the invoice (category, rate, quantity, amount)
  • Time Records — Individual time entries that fall within the invoice's billing period, showing date, professional, start/end times, category, and hours

Invoice Status Colors

ColorStatus
Blue headerPaid invoice
Amber headerOpen/unpaid invoice

Clock Time Coverage

A yellow banner appears if not all entries have clock-in/clock-out times. This happens when time was entered manually in QuickBooks rather than clocked through Workforce. Entries with clock times also undergo a duration sanity check — if the clock duration doesn't match the recorded hours (within 5 minutes), a ⚠ warning icon appears.

Exporting the Report

Print / Save as PDF

  1. Click the Print / Save PDF button above the report preview
  2. In the print dialog, select "Save as PDF" as the destination
  3. The report is formatted for print — action bars and navigation are hidden
Tip: Use Chrome's "Save as PDF" for the best formatting. The report includes print-specific CSS that hides UI elements and preserves colors.
